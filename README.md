# Proyecto Two Vite

Este proyecto fue desarrollado con React + Vite y utiliza React Router DOM y Bootstrap para construir una interfaz sencilla con varias páginas.

## Cómo iniciar el proyecto

1. Entra a la carpeta del proyecto:
   ```bash
   cd proyecto_two_vite
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre la URL que aparece en la terminal, normalmente:
   ```bash
   http://localhost:5173
   ```

4. Para generar una versión lista para producción:
   ```bash
   npm run build
   ```

5. Para previsualizar la versión compilada:
   ```bash
   npm run preview
   ```

## Estructura general

- src/components: contiene los componentes reutilizables como el encabezado, pie de página y menú.
- src/pages: almacena las páginas principales del sitio.
- src/App.jsx: define la estructura general de la aplicación y las rutas.

## Lo que aprendí al desarrollar este proyecto

- A organizar un proyecto React de forma más limpia mediante componentes reutilizables.
- A trabajar con React Router para navegar entre páginas.
- A integrar Bootstrap para dar estilo de forma rápida y consistente.
- A comprender mejor el flujo de desarrollo con Vite, desde la instalación hasta la compilación final.
- A mejorar la estructura del proyecto para que sea más fácil de mantener y expandir.
